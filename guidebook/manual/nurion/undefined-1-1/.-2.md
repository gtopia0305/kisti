# 다. 사용자 쉘 변경

&#x20;\- 누리온 시스템의 로그인 노드는 기본 쉘로 bash이 제공된다. 다른쉘로 변경하고자 할 경우 chsh 명령어를 사용한다.

```
$ chsh
```

&#x20;

&#x20;\- 현재 사용 중인 쉘을 확인하기 위해서 ech $SHELL을 이용하여 확인한다.

```
$ echo $SHELL
```

&#x20;\- 쉘의 환경설정은 사용자의 홈 디렉터리에 있는 환경설정 파일(.bashrc, .cshrc 등)을 수정하여 사용하면 된다.
