# 다. Conda Environment에 패키지 설치 및 확인

\- conda install \[패키지명] 으로 패키지를 설치할 수 있다.

\- conda 채널에 있는 패키지는 "conda install -c \[채널명] \[패키지명]" 와 같이 설치 할 수 있다.

\- 위 "2" 항목에서 생성한 conda environment 경로 아래에 패키지 들이 설치 된다.

&#x20;
